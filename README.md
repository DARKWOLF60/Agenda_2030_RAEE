# RAEE - SDG 12.5 Agenda 2030

Un progetto educativo che affronta la gestione dei Rifiuti di Apparecchiature Elettriche ed Elettroniche (RAEE) in linea con l'Obiettivo 12 di Sviluppo Sostenibile delle Nazioni Unite.

## Panoramica del Progetto

Questo sito web illustra l'importanza della corretta gestione dei RAEE e fornisce informazioni dettagliate su:

- Definizione e classificazione dei RAEE secondo la Direttiva UE 2012/19/UE
- L'importanza dello SDG 12.5 e le strategie di sostenibilità (prevenzione, riduzione, riciclaggio, riutilizzo)
- Impatto ambientale e opportunità di recupero dei materiali
- Azioni concrete per consumatori e comunità

## Struttura del Progetto

```
├── index.html          # Pagina principale (sezioni, content, struttura HTML)
├── style.css           # Stylesheet (design, layout, responsive)
├── Tipografia/         # Cartella con font personalizzati
│   ├── NeverMind-Bold.ttf
│   └── NeverMind-Light.ttf
└── Immagini/           # Cartella con immagini
    └── Immagine_Header.jpeg
```

## Caratteristiche

- **Design Responsivo**: Layout adattativo che funziona su desktop, tablet e mobile
- **Sticky Navigation**: Barra di navigazione fissa per accesso rapido alle sezioni
- **Dark Theme**: Tema scuro professionale per una migliore leggibilità
- **Grid Layout**: Sistema di griglia CSS modern per organizzazione dei contenuti
- **Typography Personalizzata**: Font custom NeverMind per un'identità visiva unica
- **Componenti Riutilizzabili**: Card, statistiche e sezioni standardizzate

## Sezioni Principali

1. **Cos'è il RAEE**: Definizione e importanza della gestione corretta
2. **SDG 12.5 - Agenda 2030**: Target specifico e 4 strategie fondamentali
3. **Categorie di RAEE**: Classificazione secondo normative UE
4. **Impatto Ambientale**: Danni e opportunità di recupero con statistiche globali
5. **Come Agire Responsabilmente**: Guide pratiche per individui e comunità

## Tecnologie Utilizzate

- **HTML5**: Struttura semantica del contenuto
- **CSS3**: Styling avanzato con flexbox e grid
- **Font personalizzati**: Web fonts per branding coerente
- **Design responsivo**: Media queries per adattamento multi-device

## Come Utilizzare

1. Clonare il repository:
   ```bash
   git clone https://github.com/tuonome/raee-sdg-12.git
   ```

2. Aprire `index.html` nel browser preferito

3. Navigare tramite il menu sticky in alto per accedere alle diverse sezioni

## Requisiti

- Browser moderno (Chrome, Firefox, Safari, Edge)
- Connessione internet (per caricamento font e immagini)
- Screen width minimo 300px per mobile
